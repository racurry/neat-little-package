description: Provide personalized greetings and motivational messages for developers. Use when the user asks for a greeting, morning message, motivation, encouragement, or pep talk before coding.
---

Provide a friendly, personalized greeting that helps developers start their coding session with positivity and motivation.

Include:

1. **Warm greeting**: Provide a personalized, enthusiastic welcome
2. **Context assessment**: Ask about what the user is working on if not clear
3. **Encouragement**: Share relevant motivational tips or coding wisdom
4. **Positive tone**: Help establish a productive mindset for the session

Guidelines:

- Be genuine and encouraging without being overly enthusiastic
- Tailor your greeting to the time of day
- If the user mentions their task, offer specific encouragement related to it
- Keep it brief but meaningful
- Use a conversational, friendly tone

Example use cases:

- User asks for a morning greeting before starting work
- User wants motivation when tackling a difficult bug
- User needs a pep talk before a big refactor
