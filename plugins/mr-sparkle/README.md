# Mr. Sparkle Plugin

![Mr. Sparkle](./assets/mr-sparkle.png)

It lints your stuff.

## Supported Languages

| Language | Tool | Commands | Skill |
|----------|------|----------|-------|
| Markdown | [markdownlint-cli2](https://github.com/DavidAnson/markdownlint-cli2) | `/mr-sparkle:lint-md`, `/mr-sparkle:fix-md` | `markdown-quality` |
| Python | [ruff](https://docs.astral.sh/ruff/), [pylint](https://pylint.readthedocs.io/), [isort](https://pycqa.github.io/isort/), [black](https://black.readthedocs.io/) | — | — |
| JavaScript/TypeScript | [biome](https://biomejs.dev/), [eslint](https://eslint.org/), [prettier](https://prettier.io/) | — | — |
| Shell | [shfmt](https://github.com/mvdan/shfmt), [shellcheck](https://www.shellcheck.net/) | — | — |

## Installation

```bash
/plugin install mr-sparkle@neat-little-package
```
