# Box Factory

![My boy is a box!](./assets/box-factory.png)

Guidance and guardrails for building Claude Code components.

## Overview

- Teaches Claude when to use skills vs agents vs hooks vs memory
- Corrects gotchas Claude consistently gets wrong (sub-agent isolation, colors, UV shebangs)
- Encodes user preferences (knowledge delta, minimal plugin.json, terse READMEs, CLI over MCP)
- Read-only component reviewer agent for validating against official specs
